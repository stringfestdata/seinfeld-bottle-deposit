# seinfeld-bottle-deposit

A fun single-page visual simulator inspired by the Seinfeld bottle deposit episode.

## What it does
- Models Michigan's 10¢ deposit return value.
- Lets you tune trip assumptions (containers, miles, MPG, gas price, extra costs).
- Includes themed vehicles (Jerry's SAAB, Jon Voight's LeBaron, Newman's mail truck, etc.).
- Shows deposit total, trip cost, net result, and a quick verdict.

## Run locally
Open `index.html` in any modern browser.
